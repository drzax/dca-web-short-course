What is a CMS?

A computer program that allows publishing, editing and modifying content as well as maintenance from a central interface.

Hosted CMS

Also called Software as a Service (SaaS)

e.g. WordPress.com, Square Space

Advantages

  • Little technical knowledge required
  • Well implemented designs
  • Highly optimised code

Disadvantages

  • Inflexible
  • Designs not unique
  • Locked in

Self Hosted & Open Source

e.g. WordPress.org, Movable Type

Advantages

  • Complete control
  • Designs can be unique
  • Flexible functionality
  • Easily extended

Disadvantages

  • Higher degree of technical knowledge required
  • Can be insecure if not updated with new releases

Proprietary & Advanced Open Source Platforms

e.g. SilverStripe, ExpressionEngine, SharePoint
