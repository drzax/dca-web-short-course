Creating markup

Goals:

Links

  • Absolute
  • Relative
  • Anchor (or in-page)

Absolute

<a href="http://example.com/page.html">Other site</a>

Relative

<a href="other-page.html">Other Page</a>

Anchor

<a href="#page-section">My Page</a>

The ID attribute

<div id="page-section">Section content</div>

CSS

Cascading Style Sheets

Basic syntax

Properties

Can be:

  • HTML element (tag) name
  • class attribute
  • ID attribute
<p id="paragraph-1" class="intro">

p {
  font-color: #FF0000;
}

#paragraph-1 {
  font-color:: #FF0000;
}

.intro {
  font-color: #FF0000;
}

p {
  font-color: #FF0000;
  font-size: 20px;
}
